# Assesment Test (Vite + Bootstrap 5)

A simple UI slicing using Vite, Bootstrap 5 and SASS.

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
  pnpm install
```

Start the server

```bash
  npm start
  pnpm start
```
